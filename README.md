🎨 Práticas de HTML e CSS – Origamid

Este repositório reúne os exercícios e práticas desenvolvidos durante os cursos de HTML e CSS da Origamid, com foco na construção de interfaces modernas, organizadas e responsivas.

Ao longo das práticas, foram aplicados conceitos essenciais como HTML semântico, CSS avançado, flexbox, grid layout, tipografia, cores e boas práticas de organização de código.

Os projetos demonstram a evolução no desenvolvimento front-end, com maior atenção à estrutura, reutilização de estilos e consistência visual.

Este repositório serve como base de estudo contínuo e consolidação de conhecimentos, além de representar um avanço na criação de layouts mais profissionais e bem estruturados.

🚀 Tecnologias utilizadas: HTML5 e CSS3
📚 Foco: Layout moderno, responsividade e boas práticas
💡 Objetivo: Evoluir na construção de interfaces e aprofundar conhecimentos em front-end
